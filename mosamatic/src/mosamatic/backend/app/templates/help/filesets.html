{% load static %} This is the main page of Mosamatic.

Its main purpose is to upload and view filesets. A fileset is a collection of files. Each fileset has a unique ID and a name. The name does not have to be unique but to distinguish filesets it's best to choose a unique name.

Uploading a single fileset
Files can be uploaded in two different ways: (1) as a single fileset and (2) as multiple filesets. To upload a single fileset, you click Choose Files and navigate to the directory containing the files you wish to upload. Please be aware, that ALL files in this directory will be uploaded to Mosamatic. You cannot upload individual files. The reason Mosamatic only supports directory upload is because this allows uploading multiple directories (e.g., each containing files for a given imaging scan), resulting in multiple filesets.

To upload files into a single fileset, make sure to enable the Store files as single fileset checkbox. Click Choose Files and select the directory containing the files in question. You can provide a name for the fileset. If you don't, a name will be generated automatically.
Once the files are uploaded (an animated icon will be shown while the upload is in progress), the fileset will appear below the Upload button. You can click the fileset name to view the contents of the fileset, i.e., the set of individual files.

To remove a fileset from Mosamatic, click the [delete] link to the right of the fileset. To remove all filesets at once, click the Delete all filesets link just below the Upload button.

Uploading multiple filesets
To upload multiple filesets at once, e.g., to upload multiple CT scans, make sure you disable the Store files as single fileset checkbox. Then click Choose Files and navigate to the TOP-LEVEL directory containing subdirectories for each scan's files. See below for the expected directory structure:

  • Top-level directory
    • Scan 1
      • File 1
      • File 2
      • ...
    • Scan 2
    • ...

The multi-fileset upload is primarily designed for upload of multiple DICOM scans. The upload procedure automatically detects which image belongs to which scan and creates a separate fileset for each scan. The name of the fileset will be based on the scan file's directory. So, in the above directory structure, the filesets will be named Scan 1, Scan 2, etc.
