Mosamatic user documentation

Welcome to the Mosamatic user documentation. Here you will find everything you need to get started with Mosamatic. If you are able to view this page, you at least have managed to get Mosamatic running.

  2. Upload AI segmentation model files

  4. Rescale DICOM images
  5. Extract muscle/fat regions from DICOM images
  6. Calculate body composition metrics
  7. Generate PNG images from results
